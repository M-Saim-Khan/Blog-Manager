**Description:**
A simple blog management API built with FastAPI, SQLModel, GraphQL, and SQLite. The application allows users to create accounts, publish posts, retrieve user and post information, and manage data through GraphQL queries and mutations.

**Features:**
- Create users
- Create blog posts
- Retrieve users and their posts
- Retrieve individual posts
- Delete users and their associated posts

**Tools Used:**
- FastAPI
- GraphQL (Strawberry)
- SQLModel
- SQLite
